# Default Arguments and Inline Functions

This repository contains practical exercises and implementations from the Algorithms & Programming Lab module on Default Arguments and Inline Functions in C++.

## Topics Covered

* Default arguments in functions
* Function calls with optional parameters
* Inline functions
* Performance benefits of inline functions
* Code optimization techniques
* Practical implementations using default arguments and inline functions

## Learning Objectives

The purpose of this repository is to understand how default arguments and inline functions work in C++. The exercises focus on simplifying function calls, improving code readability, and enhancing program efficiency through inline function implementation.

* C++
* GCC / MinGW Compiler
